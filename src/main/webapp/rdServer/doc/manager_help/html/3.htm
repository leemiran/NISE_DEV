서버 관리창

오른쪽에 보이는 창이 서버의 목록을 관리할 수 있는 관리 창이다.

여기에 관리 대상 서버를 차례로 등록하여 리스트로 편리하게 관리할 수가 있다.

한곳의 서버에 로그인 한 후 관리를 마치고 다른 쪽 서버로 가고자 할 때는 해당 서버 아이콘을 더블 클릭 하여 들어가면 된다.

 참고: 관리자 로그인

참고) Manager Port 는

-단독 서버 인 경우 : startup.bat 또는 startup.sh 스크립트의 manager.port 에 지정해준 포트번호

-서블릿 서버인 경우 : RDServer/rdagent.jsp 에서  managerport 에 지정해준 포트번호
