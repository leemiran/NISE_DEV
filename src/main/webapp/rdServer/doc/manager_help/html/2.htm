관리자 로그인

RDServer를 관리하기 위해 사용자 인증과정을 거쳐야 한다. 왼쪽 서버 트리에 등록된 각 RDServer에서 관리 하고자 하는 RDServer를 두 번 클릭하면 로그인 할 수 있도록 그림(1)과 같이 다이얼로그가 뜬다. 만약 RDServer를 처음 설치한 서버이면 id(admin) password(admin)을 만들고 자동 로그인 된다.(계정의 관리는 9. 시스템관리의 보고서 보안 사용자 추가에서 관리합니다.)
로그인 다이얼로그에서 “자동 로그인 사용”을 체크하고 로그인 하면 다음부터는 로그인 다이얼로그가 뜨지 않고 마지막으로 로그인한 아이디로 자동 로그인 하게 된다.

 

그림(1)

 

서버로 로그인 시 administrator레벨은 모든 메뉴가 활성화가 되고 user레벨은 스케줄링쪽 메뉴만 활성화가 된다.

그림 (2) 로그인 성공 시 화면

 

자동 로그인 취소는 서버 관리창에서 오른쪽 마우스 버튼을 누르면 나타나는 자동 로그인 취소 메뉴에서 할 수 있다.
