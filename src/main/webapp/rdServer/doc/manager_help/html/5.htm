툴바

Manager의 툴바는 다음과 같이 구성되어 있다. 툴바는 빠른 엑세스를 제공한다.

 

1. 등록된 스케줄링 목록보기

2. 메모리 모니터링

3. 보고서 사용자 로그보기

4. 보고서 사용자

5. 서버 업데이트

6. 서버 정보 보기

현재 운영 중인 서버 에 대한 세부 정보를 확인할 수 있다

서버 매니저의 툴바 메뉴로 제공된다

현재 연결된 RDServer가 데몬인지 서블릿버전인지를 확인할수 있고,

서버 OS 종류, 라이센스, 보고서 서비스 포트 등 기타 JVM 환경등을 볼 수 있다.

7. 연결 끊기

현재 접속 중인 서버의 연결을 끊는다.
